# Password Strength Checker

A Python script that evaluates the strength of a given password based on length, uppercase, lowercase, digits, and special characters.  
It provides a strength rating (`Strong`, `Medium`, or `Weak`) and suggestions for improvement.

---

## Features
- Checks if the password meets minimum length requirements
- Validates presence of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Returns a strength rating
- Provides suggestions to improve weak passwords
