# Password Generator

A simple Python script to generate secure random passwords.  
Ensures at least one lowercase, uppercase, digit, and special character, with customizable length and shuffled order.

---

# Features
- User can specify password length (e.g., 12, 16, etc.)
- Random password length (8–10 characters) if no input is given
- Includes lowercase, uppercase, digits, and symbols
- Guarantees at least one of each type
- Shuffles characters for extra randomness
